📋 Best Practices & Notes

⚠️ Rate Limiting

Saat ini belum ada rate limiting, namun diharapkan tidak spam request. Best practice:

  • Maksimal 60 requests per menit per IP
  • Gunakan cache untuk data yang jarang berubah
  • Batch requests jika memungkinkan
💡 Caching

Server memiliki built-in image caching. Image proxy tersedia di:

GET /img/:hash.:ext

Hash di-generate otomatis oleh server. Gunakan URL image dari response API.

🔄 CORS

API sudah enabled CORS untuk semua origin (*). Bisa diakses dari:

  • Browser (fetch/axios)
  • Mobile apps
  • Desktop applications
  • Other servers
📡 Response Format

Semua response mengikuti format standar:

{
  "status": "success" | "error",
  "data": {...} | null,
  "message": "Error message if status is error"
}
🌐 HTTPS Support

Untuk production, disarankan:

  • Deploy dengan HTTPS (Let's Encrypt)
  • Gunakan reverse proxy (Nginx/Caddy)
  • Set proper CORS origin restrictions
  • Add rate limiting middleware

💻 Code Examples

JavaScript (Fetch API)

// Get Homepage V2
fetch('http://localhost:5000/api/v2/homepage')
  .then(res => res.json())
  .then(data => {
    if (data.status === 'success') {
      console.log('Anime:', data.data);
    }
  })
  .catch(err => console.error('Error:', err));

// Search with filters
async function searchAnime() {
  const params = new URLSearchParams({
    genre: 'action',
    type: 'TV',
    order: 'popular',
    page: 1
  });
  
  const response = await fetch(`http://localhost:5000/api/v2/all-anime?${params}`);
  const data = await response.json();
  return data.data;
}

Python (Requests)

import requests

# Get Homepage
response = requests.get('http://localhost:5000/api/v2/homepage')
data = response.json()

if data['status'] == 'success':
    for anime in data['data']:
        print(f"{anime['title']} - {anime['episode']}")

# Search with filters
params = {
    'genre': 'action',
    'type': 'TV',
    'order': 'popular',
    'page': 1

response = requests.get('http://localhost:5000/api/v2/all-anime', params=params)
anime_list = response.json()['data']['animeData']

Node.js (Axios)

const axios = require('axios');

// Get anime detail
async function getAnimeDetail(slug) {
  try {
    const response = await axios.get(`http://localhost:5000/api/v2/anime/${slug}`);
    return response.data.data;
  } catch (error) {
    console.error('Error:', error.message);
    return null;
  }
}

// Get with multiple genres
async function getAnimeByGenres(genres) {
  const params = new URLSearchParams();
  genres.forEach(genre => params.append('genre', genre));
  
  const response = await axios.get('http://localhost:5000/api/v2/all-anime', { params });
  return response.data.data.animeData;
}

// Usage
getAnimeByGenres(['action', 'fantasy']).then(anime => console.log(anime));

⚠️ Error Handling

HTTP Status Codes

Status Code Meaning Action
200 Success Request berhasil, data tersedia
400 Bad Request Parameter tidak valid, check query params
404 Not Found Anime/Episode tidak ditemukan
500 Internal Server Error Error scraping, retry setelah beberapa saat
503 Service Unavailable Source website down, tunggu beberapa menit

Error Response Format

{
  "status": "error",
  "message": "Anime not found",
  "data": null
}

Common Errors

Anime/Episode Not Found

Check slug, pastikan format benar dan anime masih tersedia

Timeout Error

Source website lambat, retry dengan backoff

Scraping Failed

HTML structure berubah atau website maintenance
